## JavaScript Generator for Random Characters

Here's a script you can run in your browser console or embed into your webpage to generate random characters from this range:

```js
function getRandomLatinExtendedEChar() {
  const start = 0xAB30;
  const end = 0xAB61; // Adjust to last code point you want
  const codePoint = Math.floor(Math.random() * (end - start + 1)) + start;
  return String.fromCodePoint(codePoint);
}
```
